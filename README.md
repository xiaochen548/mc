# 小晨的Minecraft资源站

一个简约现代风格的Minecraft资源分享网站，包含整合包、启动器、模组和地图等下载资源。

## 功能特点

- 响应式设计，适配各种设备
- 丰富的动画效果，提升用户体验
- 清晰的资源分类，方便用户查找
- 简洁的界面设计，易于使用
- 完整的下载功能模拟

## 技术栈

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript
- AOS 动画库
- GSAP 动画库

## 本地运行

1. 克隆或下载本项目
2. 直接打开 `index.html` 文件即可在浏览器中查看

## 项目结构

```
minecraft-resource-site/
├── index.html          # 主页面
└── README.md           # 项目说明
```
